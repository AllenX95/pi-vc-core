# Memory Capture Rules

1. Capture strong user signals by default.
2. Do not capture ordinary agent output.
3. Do not capture parsed document text as memory.
4. Do not capture search results or OCR text as long-term memory.
5. Agent-generated judgments require user confirmation, correction, or adoption before capture.
6. Sensitive-looking content requires confirmation or summary-only capture.
7. Dream updates require proposal, approval, and commit.
