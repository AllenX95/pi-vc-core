description: Create follow-on VC workflow skills that use pi-vc-core workspace, office, and memory APIs.
---

When creating new VC workflow skills for this package:

1. Use workspace APIs for all project file paths and artifacts.
2. Use office APIs for Office/PDF parsing and patching.
3. Use memory APIs only for user-confirmed signals.
4. Keep each skill narrowly scoped and testable.
